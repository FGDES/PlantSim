Nominal controller design for "bottling plant"

This is the exmple from the technical report distributed
as documentation of the FtcPlant simulator.

To run the supervisor by simfaudes, you may use provided sim-file

> cd FTCPLANT_EXAMPLES/FtcNominal
> LIBFAUDES_BIN/simfaudes -dr -d ../ftcsuper.dev  ftcsuper.sim

(substitute FTCPLANT_EXAMPLES and LIBFAUDES_BIN according to
you set-up)
